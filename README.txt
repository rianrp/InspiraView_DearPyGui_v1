# InspiraView com Dear PyGui (v1+)

Este projeto usa Dear PyGui versão moderna (1.x ou superior) para criar uma interface flutuante com imagem carregável.

## Recursos:
- Sempre por cima;
- Controle de opacidade;
- Suporte a PNG, JPG, WebP;
- Interface moderna e leve.

## Instalar dependências:
```bash
pip install dearpygui pillow
```

## Executar:
```bash
python main.py
```
